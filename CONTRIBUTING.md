# Contributing to CullSnap

Thank you for considering contributing to CullSnap!

## How to Contribute

1.  **Fork the repository**: Create your own copy of the project on GitHub.
2.  **Clone your fork**:
    ```bash
    git clone https://github.com/YOUR_USERNAME/cullsnap.git
    cd cullsnap
    ```
3.  **Create a branch**: `git checkout -b feature/my-new-feature`
4.  **Make your changes**.
5.  **Test locally**:
    ```bash
    make run
    ```
6.  **Format your code**: Please ensure `go fmt` is run before submitting a PR.
    ```bash
    go fmt ./...
    ```
7.  **Commit and Push**:
    ```bash
    git commit -m "Add amazing feature"
    git push origin feature/my-new-feature
    ```
8.  **Open a Pull Request**.

## Reporting Bugs

Please include OS version, steps to reproduce, and any relevant logs.
